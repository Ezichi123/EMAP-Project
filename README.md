# EMAP Dataset Repository

This repository contains the **train** and **validation** data of the EMAP dataset. The **test** set is not provided.

## Dataset Overview

- **Training Set**: The data of 87 participants comprises the training set.
- **Validation Set**: The data for 29 participants comprises the validation set.
- **Test Set**: The test set is not provided.

All participants have watched 24 trials, with the following exceptions for the training set:

### Participants with less than 24 trials in the train set:
- **Participant 34**: 23 trials
- **Participant 36**: 23 trials
- **Participant 62**: 12 trials
- **Participant 63**: 16 trials
- **Participant 65**: 12 trials
- **Participant 88**: 12 trials

The training set is further split into 3 folders:
- **P1-40**
- **P41-80**
- **P81-90**

This splitting is due to GitHub's restriction of allowing only 1000 files per directory.

## How to Use the Dataset

To have a complete set of the training data, please move all data into the **train** folder.
